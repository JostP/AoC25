Vhod je seznam junction box-ov v 3D prostoru.

Najprej se izračuna evklidska razdalja med vsemi pari junction box-ov.  
Najkrajše povezave se izvedejo in junction box-i, ki so povezani, dobijo isto oznako kroga.  

V Part 1 se izvede 1000 najkrajših povezav in nato se zmnožijo tri največje skupine.  

V Part 2 se izvede vse povezave in ob vsakem združevanju se izpiše, kateri box-i so povezani. Na koncu se zmnožijo tri največje skupine.
