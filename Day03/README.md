Program bere vhod iz datoteke input.txt, kjer vsaka vrstica predstavlja eno banko baterij.

V prvem delu se za vsako vrstico poiščeta dve največji bateriji glede na vrstni red v nizu in iz njiju sestavi največje možno dvomestno število.

V drugem delu se izbere več baterij. Postopek poteka od desne proti levi, kjer se vsakič izbere največja možna baterija, pri tem pa se omeji območje iskanja z že izbranimi baterijami. Tako nastane največje možno število za posamezno banko.

Končni rezultat za vsak del je vsota vseh vrednosti po bankah.
