Vhod predstavlja mrežo papirnih rol, kjer znak @ pomeni rolo, pika pa prazen prostor.

V prvem delu program preveri vsako rolo posebej. Rola je dostopna, če ima v osmih sosednjih poljih manj kot štiri druge role. Program prešteje vse take role.

V drugem delu se role odstranjujejo postopoma. Vsakič, ko je rola dostopna, se odstrani iz mreže. Postopek se ponavlja, dokler v enem prehodu ni odstranjena nobena rola več. Program šteje vse odstranjene role.

Rezultat prvega in drugega dela se izpišeta ločeno.
