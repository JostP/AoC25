Vhodna datoteka vsebuje najprej razpone svežih ID-jev, nato prazno vrstico in nato seznam ID-jev sestavin.

V prvem delu program preveri vsak ID posebej in ugotovi, ali spada v katerikoli razpon. Če spada, se šteje kot svež.

V drugem delu program najprej obdela razpone tako, da odstrani prekrivanja in podvojitve. Razponi, ki so vsebovani v drugih, se izbrišejo ali skrajšajo. Na koncu se sešteje velikost vseh preostalih razponov, kar predstavlja število vseh možnih svežih ID-jev.

Rezultata prvega in drugega dela se izpišeta ločeno.
